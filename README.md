# QIQI.EplOnCpp.GUI
命名空间：`QIQI.EplOnCpp.GUI`  

# 介绍
EplOnCpp是基于Cpp构建的Epl语言编译环境，用于支持x64程序编写及实现特殊功能  
本项目为EplOnCpp中的GUI调用部分

# 编译
使用NuGet安装缺失包，然后一般编译即可  
注意：**必须VS2017或更高版本，使用了最新语言特性** *（不过运行环境.NET Framework 4.0即可，支持XP）*
